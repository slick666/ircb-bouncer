# ircb-bouncer
Simple dockerization of the ircb bouncer

## Setting up your Dev environment
### Prerequsite
 * System with Docker
 * Python 3.4+

### Steps
 1. Setup VirtualEnv
 2. Install Docker-compose
 3. call docker-compose up, and grab some coffee
